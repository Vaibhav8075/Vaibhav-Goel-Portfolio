# Vaibhav Goel's Portfolio Website

---

## To run the app follow the steps:

```bash
npm install
```

Once setup is complete (for localhost):

```bash
npm run dev
```

Final production build command:

```bash
npm run build
```

---

## Tech Stack

- React 19
- Vite 7
- Three.js + React Three Fiber + Drei
- Framer Motion
- Tailwind CSS
